General Genetic Algorithm
f(x) is the fitness function
1. Generate random population = n
2. Evaluate f(p) for each p in n
3. For all p in n
    4. Select pairs of p such that the sum of all { f(pA) - f(pB) } is minimized
    5. With some probability C, cross over pairs of p to form new offspring.
    6. With some probability M, mutate new offspring at each locus.
    7. Place new offspring in a new population
8. Use newly generated population for a further run of algorithm
9. If the goal is met return this solution else go to step 2 with new population.

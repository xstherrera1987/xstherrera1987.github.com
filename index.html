General Genetic Algorithm
f(x) is the fitness function
1. Generate random population := n
2. Evaluate f(p) for each p in n
3. For all p in n
    4. Select pairs of p such that the sum of all f(pA) - f(pB) is minimized
    5. With some probability C, cross over pairs of p to form new a offspring.
    6. With some probability M, mutate it at each locus.
    7. Place it in a new population for next iteration
8. If the goal is met return this solution (the fittest member in the new population) otherwise return to step 2 with the new population.
